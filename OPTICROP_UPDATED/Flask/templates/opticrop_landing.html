About the project

OptiCrop is a machine learning-powered web application built to optimize agricultural production. By analyzing nitrogen, phosphorous, potassium, temperature, humidity, pH, and rainfall, the system recommends the crop best suited to a field's current conditions — helping farmers, researchers, and policymakers make data-driven decisions that improve yield, sustainability, and resource efficiency.

Model

Trained & Compared

KNN, Logistic Regression, Decision Tree, Random Forest, and K-Means evaluated to select the best performer.

Pipeline

Clean, Persisted Data

Preprocessed for missing values and scale, with the trained model stored as a reusable .pkl file.

Delivery

Instant, in the Browser

A Flask backend serves real-time crop recommendations through a simple, responsive web form.
